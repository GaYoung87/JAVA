## 06_클래스

### [ 클래스 선언 ]

- public 접근제한자는 파일 이름과 동일한 이름의 클래스 선언에만 붙일 수 있다.

```java
public class Car {}

public class Car {} // 컴파일 에러
class Tire {}
```

--------------------------

### [ 필드 ]

- 변수 vs 필드
  - 변수 : 자신이 선언된 곳에서만 사용 가능
  - 필드 : 생성자, 메소드에서 사용 가능 

--------------------

### [ 생성자 ]

- 변수 vs 필드
  - 변수 : 자신이 선언된 곳에서만 사용 가능
  - 필드 : 생성자, 메소드에서 사용 가능 

------------------------

```java
public Car() {
	public String car;
    
    public Car(String car) {
        this.car = car;
    }
    
    public getCar() {
        return car;
    }
}
```
